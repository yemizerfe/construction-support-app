# Construction Support Application (In Progress)

A project management system designed to help construction companies track project timelines and budgets.  
The goal is to reduce delays and prevent early budget overruns.

 Planned Features
- Task timeline tracking
- Budget monitoring and alerts
- Progress reporting
- Role-based access control

 Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Node.js, Express.js
- Database: PostgreSQL

 Purpose
Construction companies often struggle with meeting deadlines and staying within budget.  
This application aims to solve those challenges using data-driven tracking and planning tools.
